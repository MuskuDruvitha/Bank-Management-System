Bank Management System

This is a Java-based Bank Management System developed using Java Swing and MySQL. 
It simulates basic banking operations such as login, account creation, deposit, withdrawal, fast cash, PIN change, and mini statement.

Technologies Used
- Java (Swing, AWT)
- MySQL
- JDBC

Project Files

Login.java  
Handles user login using account number and PIN.

Signup.java, Signup2.java, Signup3.java  
Used for new account registration and storing user details.

Transaction.java  
Main menu to access banking operations.

Deposit.java  
Allows users to deposit money.

Withdrawl.java  
Allows users to withdraw money.

FastCash.java  
Provides quick withdrawal options.

MiniStatement.java  
Displays recent transactions.

Pin.java  
Allows users to change their PIN.

conn.java  
Manages database connection using JDBC.

DataBase:
atmbank.sql  
Contains the MySQL database structure.
